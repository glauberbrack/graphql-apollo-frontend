## :notebook: Running

```bash
# Clone the project
$ git clone https://github.com/glauberbrack/graphql-apollo-frontend

# Go to the folder
$ cd graphql-apollo-frontend

# Install all the depencies
$ yarn install

# Run the project
$ yarn start
```
